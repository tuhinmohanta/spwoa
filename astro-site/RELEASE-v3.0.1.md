## v3.0.1 — Branding polish: TM favicon rollout

This patch release delivers a complete, professional favicon/icon set aligned with the site’s visual identity.

### Highlights
- Switched active favicon to the new **TM monogram**
- Added standard browser favicon PNGs: **16x16** and **32x32**
- Added **apple-touch-icon** for iOS home screen support
- Added **Android Chrome icons** (192x192 and 512x512)
- Regenerated **favicon.ico** as a multi-size icon asset
- Included design/reference files for future favicon iterations

### Why this matters
- Stronger personal brand presence in tabs and bookmarks
- Cleaner cross-platform icon coverage (desktop + mobile)
- Better consistency with the site’s gold-accent professional aesthetic
